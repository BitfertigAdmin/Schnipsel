# NPM

## Package erstellen

Zuerst einen Account anlegen auf [npmjs.com](https://www.npmjs.com/) und übers Terminal/Bash einloggen.

### Login

```bash
 npm login
```

### Package erstellen und hochladen

```bash
npm publish --access public
```

## Package löschen
```bash
npm unpublish <package-name> -f
```
